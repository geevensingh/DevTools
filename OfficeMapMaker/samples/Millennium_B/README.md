# samples\Millennium_B

This folder is the regression sample for `OfficeMapMaker`. Drop the following
files here to use it end-to-end:

| File | Purpose |
|---|---|
| `map.png` | The Millennium B first-floor map |
| `assignments.csv` | The `Full Name, Office number, Team` rows (anonymized before committing) |
| `calibration.json` | Generated by `officemapmaker calibrate`, then hand-reviewed and checked in as the **golden calibration** for this map |
| `teams.json` *(optional)* | Explicit team color overrides |

> **Privacy:** real names of Microsoft employees must **never** be committed.
> When checking in a sample assignment file, replace each `Full Name` with a
> synthetic name (e.g., `Last, First` placeholders) but keep the original team
> distribution and office assignments so the layout pass still exercises the
> name-fit ladder realistically.
