# DiffViewer has moved

DiffViewer lived here as part of the [DevTools] personal toolkit monorepo
until May 2026, when it was extracted into its own home:

**https://github.com/geevensingh/DiffViewer**

Downloads (single-file `.exe` for Windows 10+):
**https://github.com/geevensingh/DiffViewer/releases**

The full commit history before the extraction is preserved in the new repo
via `git filter-repo`. The same commits still exist *in this repo's history*
too — anything you had a `geevensingh/DevTools/blob/<old-sha>/DiffViewer/...`
bookmark for still resolves to its original content because git doesn't
forget old commits. New work happens in the new repo only.

[DevTools]: https://github.com/geevensingh/DevTools
